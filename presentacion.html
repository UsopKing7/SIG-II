2026

Problemática

Falta de visualización georreferenciada de la red de teleférico

Dificultad para gestionar líneas, estaciones, cables y postes de forma centralizada

No existe un sistema CRUD con datos espaciales para administrar la red

La información de la infraestructura está dispersa y no hay un mapa unificado.

Objetivos

General

Desarrollar un SIG web para visualizar y gestionar la red de teleférico universitario.

Específicos

Modelar geometrías espaciales con PostGIS

Exponer API REST con NestJS

Visualizar mapa 3D con Mapbox GL

Arquitectura General

Backend  ·  NestJS + Clean Architecture + TypeScript
Prisma ORM
